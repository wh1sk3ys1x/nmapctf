# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project Overview

Nmap Continuous Testing Framework (nmapctf) — a platform for running scheduled and ad-hoc nmap scans against network assets, tracking results, and generating reports. Packaged as a Docker container.

## Key Requirements (from outline.txt)

- Run any type of nmap scan against defined assets
- Schedule recurring scans and run one-off scans
- Store and track scan results over time
- Generate reports from scan results
- Deployable as a Docker container

## Workflow

- Commit all changes after each completed step. Do not batch multiple steps into one commit.
- Keep documentation (design spec, CLAUDE.md, README, etc.) updated as changes are made. Documentation updates should be part of the same commit as the related code changes.
